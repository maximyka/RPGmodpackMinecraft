# Лаунчеры:
* launcher.exe - для Windows (Рекомендуется: Создать новую папку и запустить оттуда, так как сборка установится рядом с лаунчером)
* launcher (без .exe) - для MAC

---

## Правила обновления сборки:
1) generate_pack.py - запускать при любом изменении - это индексация для лаунчера! 
2) большие файлы >100мб гит не грузит - загружаем в Releases и вписываем ссылку в .temp в папке mods (по примеру катаклизма) 
3) лаунчер синхронизирует "mods", "kubejs", "datapacks", "resourcepacks", "config", "options.txt"
